提供機能

ShirumaScripts が提供する主な機能：

1. Script Library（スクリプト一覧）

Roblox の人気ゲーム向けスクリプトをカテゴリ別に整理して掲載。必要なスクリプトをすぐに探せる。

2. One‑Click Copy（ワンクリックコピー）

各スクリプトにはコピー専用ボタンを設置。ワンクリックでコードを取得してすぐに使える。

3. Save user information (ユーザー情報保存)

ユーザーが、タブを閉じたりサイトを再読み込みをしてもユーザーの情報が自動的に保存されます。

4. User Feedback（いいね・コメント）

スクリプトごとに「いいね」「コメント」が可能。ユーザーの評価で人気スクリプトが一目でわかる。

5. Script Submission（スクリプト投稿）

ユーザー自身がスクリプトを投稿できる機能。コミュニティで共有し合える場を提供。

6. Safe & Clean Scripts（安全性チェック）

悪質なコードを排除し、安全性を重視したスクリプトのみ掲載。安心して利用できる環境を維持。
